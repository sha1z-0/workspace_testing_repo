# Logo Setup Instructions

## Required Logo Files

Please save your logo images to this `public` folder with the following names:

### 1. Full Logo (with text)
- **Filename:** `finova-logo-full.png`
- **Usage:** Login page, email signatures
- **Recommended size:** 1000x250px or similar (width should be ~4x the height)
- **Format:** PNG with transparent background

### 2. Icon/Mark Only
- **Filename:** `finova-icon.png`
- **Usage:** Sidebar, browser tab (favicon), mobile app icon
- **Recommended size:** 512x512px (square)
- **Format:** PNG with transparent background

### 3. Favicon (Optional - will auto-generate from icon)
- **Filename:** `favicon.ico`
- **Size:** 32x32px
- **Format:** ICO format

## Steps to Add Your Logos

1. Save the two logo images you provided as:
   - `finova-logo-full.png` (the horizontal logo with "FINOVA" text)
   - `finova-icon.png` (just the shield/badge icon)

2. Place both files in this `public` folder

3. The code has already been updated to use these files
